Preguntas para responder

1.Dar un par de ejemplos de uso de etiquetas meta

Algunos ejemplos de etiquetas meta son: "meta charset="UTF-8"" para definir la codificación de caracteres utilizada en la página y "meta name="description" content="Descripción de la página"" para proporcionar una descripción breve de la página.


2.Nombrar al menos 2 atributos aplicables a la etiqueta HTML

Algunos de los atributos aplicables a la etiqueta HTML son el charset, que establece el juego de caracteres utilizado en el documento, y el name, que proporciona un nombre para el elemento que puede ser utilizado por otras etiquetas o scripts.


3.Como se relaciona un header http con la etiqueta head de HTML?

El header HTTP se utiliza para enviar información adicional en una solicitud o respuesta HTTP, mientras que la etiqueta head de HTML se utiliza para incluir información sobre el documento, como el título, los enlaces a hojas de estilo y scripts, y las metaetiquetas.


4. Que es UTF8?

UTF-8 es un conjunto de caracteres que permite representar un amplio rango de caracteres de diferentes idiomas y scripts.


5. Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?

El atributo HTML que se utiliza para aplicar estilo es el atributo "style", que se aplica a cualquier etiqueta HTML y permite establecer estilos en línea.


6. Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?

Se indica que un enlace debe abrirse en una nueva pestaña utilizando el atributo "target" con el valor "_blank" en la etiqueta de anclaje (a).


7. Explicar cuando usarían etiquetas semánticas y con que propósito.

Las etiquetas semánticas se utilizan para proporcionar información adicional sobre el contenido de un documento HTML, lo que puede ser útil para la accesibilidad y el SEO.


8. ¿Que atributos de estilo básicos aplicarían a una tabla? Explicar.

Algunos de los atributos de estilo básicos que se aplican a una tabla son el ancho de borde, la alineación de texto y el espaciado entre celdas.


9. ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

La etiqueta body de un documento HTML contiene todo el contenido visible del documento, mientras que el body del protocolo HTTP es una parte de la cabecera HTTP que contiene información adicional sobre la solicitud o respuesta.


10. ¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?

Los atributos que se utilizan para agrupar celdas de una tabla son "rowspan" y "colspan", que se aplican a las etiquetas de celda (td) y encabezado (th).



Cuerpo Principal del Contenido

La etiqueta main es en HTML5 una etiqueta semántica. Esto quiere decir que tiene que ver con el significado de lo contenido dentro de ella. Define el contenido principal de la pagina. Unicamente puede haber una sola etiqueta como esta en todo el documento. El contenido escrito aqui dentro no se repite en el resto del sitio (todo lo contrario a los encabezados de pagina, los menues de navegacion y pie de pagina). Es usado por buscadores para indexar. Conviene usar aqui textos que ayuden a encontrar la pagina luego de procesos de indexacion de los buscadores

Seccion etiquetas semanticas estructurales mas importantes:


  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADDRESS

Las estiquetas semanticas estructurales estanr relacionadas directamente con el significado del contenido y no con los estilos ni las posiciones dentro del documento.


Reglas para etiquetas semanticas:

  • Los distintos tipos de contenido se pueden asociar a las etiquetas SECTION
  • Las unidades de contenido se asocian a las etiquetas ARTICLE. O sea que un section anida múltiples ARTICLES
  • Las etiquetas HEADER y FOOTER pueden ser anidadads dentro de otras HEADER o FOOTER.
  • La etiqueta NAV debe ser única en el documento y corresponde a los elementos de navegación del documento
  • Reservar etiqueta h1 para usar dentro de etiquetas top level, como main, header o footer.
